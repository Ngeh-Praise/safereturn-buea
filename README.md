# SafeReturn Buea

A dedicated humanitarian response and coordination platform for missing person incidents in the Buea community.

## Tech Stack
- Frontend: React (Vite)
- Styling: Tailwind CSS v4
- Routing: React Router DOM

## Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/safereturn-buea.git](https://github.com/YOUR_USERNAME/safereturn-buea.git)
   cd safereturn-buea
   npm install
   npm run dev
